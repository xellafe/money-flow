## Spacing Scale

Standard 8-point scale. All values are multiples of 4.

| Token | Value | Usage in Phase 5 |
|-------|-------|-----------------|
| xs | 4px | Gap between Tag icon and badge text; gap between sort icon and column label |
| sm | 8px | Horizontal padding inside filter chips; vertical padding on compact rows |
| md | 16px | Horizontal padding on rows and sticky header; left padding on filter bar |
| lg | 24px | Vertical padding inside filter bar section; gap between filter controls |
| xl | 32px | Section gap between filter bar and list container |
| 2xl | 48px | Empty state vertical padding top/bottom |
| 3xl | 64px | Not used in Phase 5 |

**Exceptions:**
- Row `min-height`: 56px — ensures touch-friendly delete target and comfortable scan density
- Delete button touch area: 32×32px minimum (`p-1` wrapper around Trash2 icon at `size={16}`)
- Filter chip height: 24px (`py-0.5 px-2` — compact read-at-a-glance strip)
- Sticky header height: 36px (`py-2` above/below text-xs labels)

|------|------|----------------|--------|-------------|-------|
| Meta | 12px | `text-xs` | 500 (medium) `font-medium` | 1.3 | Column header labels (uppercase, tracked); row date string; badge text; filter chip text; pagination counter |

## Interaction Contracts

### Sort State Machine
```
Initial state: sortColumn='date', sortDirection='desc'

Click column header:
  IF column ≠ sortColumn → set sortColumn=column, sortDirection='asc', currentPage=1
  IF column === sortColumn AND sortDirection='asc' → set sortDirection='desc', currentPage=1
  IF column === sortColumn AND sortDirection='desc' → reset sortColumn='date', sortDirection='desc', currentPage=1
```
Visual feedback: ArrowUpDown (inactive) → ChevronUp (asc) → ChevronDown (desc) → ArrowUpDown (reset)

### Search Debounce
- Input `value` is bound to local `inputValue` state (immediate — no lag in typing)
- After 200ms of no keystrokes: call `setSearchQuery(inputValue)` on parent
- Clearing search: immediate (no debounce on clear)
- Search icon shows as `text-gray-400`; when input is non-empty: show X button (`<button onClick={() => { setInputValue(''); setSearchQuery(''); }}><X size={14} /></button>`) replacing search icon, `text-gray-500 hover:text-gray-700`

### Filter Chip Dismissal
- Category chip ×: calls `setTransactionsCategoryFilter(null)` + resets `currentPage=1`
- Search chip ×: clears search input value + calls `setSearchQuery('')` + resets `currentPage=1`
- Year/month chips: read-only (no × button) — controlled by AppHeader PeriodSelector only

### Delete Flow
- Trash2 icon click → calls `setConfirmDelete(tx)` (opens ConfirmModal via useModals)
- ConfirmModal handles actual deletion (Phase 6 scope for modal styling)
- On row: `text-gray-300 hover:text-expense-500 transition-colors duration-150`

### Row Animation
- `AnimatePresence initial={false} mode="popLayout"` wraps row list
- Each `motion.div key={tx.id}`: enter `opacity: 0 → 1, y: -4 → 0` over 150ms
- Exit `opacity: 1 → 0, y: 0 → 4` over 150ms
- `layout` prop on motion.div enables smooth reorder on sort change
- Skip animation on initial mount (`initial={false}`)

- Search input: `aria-label="Cerca transazioni"` (in addition to placeholder)
